# AbfScraper 使用说明

## 一、准备工作

### 1. 获取文件
需要以下两个文件（由开发方提供）：

```
AbfScraper.dll
HtmlAgilityPack.dll
```

将两个文件放在同一目录下。

### 2. 项目引用
在你的 .NET 项目中添加对 `AbfScraper.dll` 和 `HtmlAgilityPack.dll` 的引用。

> 支持 .NET Standard 2.0 及以上（.NET Framework 4.6.1+、.NET Core 2.0+、.NET 5/6/7/8）

### 3. 在代码顶部引入命名空间
```vb
Imports AbfScraper
```

---

## 二、使用步骤

```
第一步：创建 AbfClient 实例
第二步：调用 Login() 登录
第三步：调用 Search() 搜索，获取结果二维数组
第四步：读取返回数组中的字段值
```

---

## 三、方法说明

### 3.1 创建客户端 `New AbfClient()`

```vb
' 最简创建（不翻译，词典文件自动存放在程序目录）
Dim client As New AbfClient()

' 使用百度翻译（推荐国内使用）
Dim client As New AbfClient(
    translationProvider:=TranslationProvider.Baidu,
    baiduAppId:="你的AppId",
    baiduSecretKey:="你的SecretKey")

' 使用百度翻译，并指定词典文件路径
Dim client As New AbfClient(
    dictPath:="C:\MyApp\trans_dict.txt",
    translationProvider:=TranslationProvider.Baidu,
    baiduAppId:="你的AppId",
    baiduSecretKey:="你的SecretKey")
```

**构造函数参数：**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `dictPath` | String | `""` | 翻译词典缓存文件路径；留空则自动使用程序目录下的 `trans_dict.txt` |
| `translationProvider` | TranslationProvider | `MyMemory` | 翻译后端：`MyMemory`（无需Key）或 `Baidu`（需AppId+SecretKey） |
| `baiduAppId` | String | `""` | 百度翻译 AppId（仅 Baidu 模式需要） |
| `baiduSecretKey` | String | `""` | 百度翻译 SecretKey（仅 Baidu 模式需要） |

---

### 3.2 登录 `Login()`

```vb
Dim ok As Boolean = client.Login("your@email.com", "your_password")
If Not ok Then
    MsgBox("登录失败")
    Return
End If
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `username` | String | 登录邮箱 |
| `password` | String | 密码 |
| **返回值** | Boolean | `True` = 登录成功 |

> ⚠️ 必须先登录，再调用 Search()，否则会抛出异常。

---

### 3.3 搜索 `Search()`

```vb
Dim arr As String(,) = client.Search(
    model:="NU324-E-TVP2",        ' 轴承型号，允许传 "" 表示不限型号
    brand:="FAG",                  ' 品牌，传 "" 表示不限品牌
    matchMode:=1,                  ' 可选：匹配模式，默认 1
    timeoutSeconds:=120,           ' 可选：超时秒数，默认 120
    maxResults:=10,                ' 可选：最多返回几条，0 = 不限，默认 0
    enableTranslation:=True,       ' 可选：是否翻译为中文，默认 False
    imageSavePath:="C:\img\"       ' 可选：图片下载目录，留空不下载
)
```

**参数说明：**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `model` | String | — | 轴承型号，如 `"NU324-E-TVP2"`；传 `""` 表示不限型号 |
| `brand` | String | — | 品牌，如 `"FAG"`；传 `""` 表示不限品牌 |
| `matchMode` | Integer | `1` | `0`=包含 / `1`=开头匹配 / `2`=完全匹配 |
| `timeoutSeconds` | Integer | `120` | 整次 Search() 的总超时秒数（含所有分页+详情页请求）；结果多时请调大，`0` 或负数表示无限等待 |
| `maxResults` | Integer | `0`（不限） | 最多返回多少条；实际结果不足时返回实际数量 |
| `enableTranslation` | Boolean | `False` | `True` 时将英文字段自动翻译为中文 |
| `imageSavePath` | String | `""`（不下载）| 图片存储目录路径，不存在时自动创建 |

---

### 3.4 返回值说明

`Search()` 返回一个二维字符串数组 `String(,)`：

- **无结果时**：`arr.GetLength(0) = 0`
- **有结果时**：行下标从 `0` 开始，列下标 `0`~`21`

| 列下标 | 字段名 | 格式 / 示例 |
|:---:|---|---|
| `arr(i, 0)` | 总条数 | `"5"` — 本次搜索命中的总结果数，每行相同 |
| `arr(i, 1)` | 结果型号 | `"NU324E.TVP2 FAG"` |
| `arr(i, 2)` | 简述 | `"Single-row cylindrical roller bearing..."` |
| `arr(i, 3)` | 原价\|折扣价 | `"€ 82,31|€ 75,00"` 或 `"€ 82,31|"` |
| `arr(i, 4)` | 库存 | `"16 pcs"` / `"0 pcs"` |
| `arr(i, 5)` | 等效型号 | `"7312B FAG|0 pcs|€ 50,00||7312-B FAG|0 pcs||"` （每4字段一条：型号\|库存\|原价\|折扣价） |
| `arr(i, 6)` | 产品描述（英\|中） | `"The NU324...|该NU324..."` |
| `arr(i, 7)` | 也被称为 | `"NU324-E-TVP2"` |
| `arr(i, 8)` | 类别（英\|中） | `"Cylindrical Roller Bearing|圆柱滚子轴承"` |
| `arr(i, 9)` | 内径 (mm) | `"120"` |
| `arr(i, 10)` | 外径 (mm) | `"260"` |
| `arr(i, 11)` | 宽度 (mm) | `"55"` |
| `arr(i, 12)` | 重量 (kg) | `"13.3"` |
| `arr(i, 13)` | 钻孔（英\|中） | `"C - Cylindrical Bore|圆柱孔"` |
| `arr(i, 14)` | 密封（英\|中） | `"No Seal|无密封件"` |
| `arr(i, 15)` | 保持架类型（英\|中） | `"P - Plastic Molded Cage|塑料保持架"` |
| `arr(i, 16)` | 外部改装（英\|中） | `"No External Modification|无外部改装"` |
| `arr(i, 17)` | 径向内部游隙（英\|中） | `"Cn Normal Internal Play|Cn正常游隙"` |
| `arr(i, 18)` | 精度（英\|中） | `"Standard Precision|标准精度"` |
| `arr(i, 19)` | 热稳定（英\|中） | `"No Heat Stabilization|无热稳定处理"` |
| `arr(i, 20)` | 图片文件名 | `"NU324ETVP2FAG-1-1.png,NU324ETVP2FAG-1-2.png"` |
| `arr(i, 21)` | 后缀描述 | `"Increased Load Capacity|增强承载能力|X-Life|X-Life（超长寿命）"` （每2字段一条：英文\|中文） |

> **通用格式规则：**
> - 所有字段无数据时统一返回 `"|"`，不返回空字符串
> - 含英中文的字段格式固定为 `"英文|中文"`，`enableTranslation=False` 时中文部分为空（`"英文|"`）
> - 等效型号每4字段为一条记录，循环排列

## 四、完整采集示例

```vb
Imports AbfScraper

Module Example
    Sub Run()
        Try
            Using client As New AbfClient(
                    dictPath:="C:\MyApp\trans_dict.txt",
                    translationProvider:=TranslationProvider.Baidu,
                    baiduAppId:="你的AppId",
                    baiduSecretKey:="你的SecretKey")

                ' 第一步：登录
                If Not client.Login("your@email.com", "your_password") Then
                    MsgBox("登录失败")
                    Return
                End If

                ' 第二步：搜索
                Dim arr = client.Search(
                    model:="NU324-E-TVP2",
                    brand:="FAG",
                    matchMode:=1,
                    timeoutSeconds:=120,
                    maxResults:=5,
                    enableTranslation:=True,
                    imageSavePath:="C:\MyApp\images\")

                ' 第三步：读取结果
                Dim total = arr.GetLength(0)
                If total = 0 Then
                    MsgBox("无结果")
                    Return
                End If

                MsgBox("搜索命中总数：" & arr(0, 0) & "，已返回：" & total & " 条")

                For i = 0 To total - 1
                    Dim resultName  = arr(i, 1)   ' 结果型号
                    Dim price       = arr(i, 3)   ' 原价|折扣价
                    Dim stock       = arr(i, 4)   ' 库存
                    Dim description = arr(i, 6)   ' 产品描述（英文|中文）
                    Dim category    = arr(i, 8)   ' 类别（英文|中文）
                    Dim innerDiam   = arr(i, 9)   ' 内径
                    Dim outerDiam   = arr(i, 10)  ' 外径
                    Dim width       = arr(i, 11)  ' 宽度
                    Dim weight      = arr(i, 12)  ' 重量
                    Dim imageNames  = arr(i, 20)  ' 图片文件名（逗号分隔）
                    Dim equivalents = arr(i, 5)   ' 等效型号

                    ' 拆分价格
                    Dim priceParts  = price.Split("|"c)
                    Dim formerPrice = priceParts(0)               ' 原价
                    Dim discPrice   = If(priceParts.Length > 1, priceParts(1), "")  ' 折扣价

                    ' 拆分产品描述中英文
                    Dim descParts   = description.Split("|"c)
                    Dim descEn      = descParts(0)
                    Dim descZh      = If(descParts.Length > 1, descParts(1), "")
                Next

            End Using
        Catch ex As Exception
            MsgBox("错误：" & ex.Message)
        End Try
    End Sub
End Module
```

## 五、单独使用翻译功能

不采集数据，仅翻译时，直接使用 `AbfTranslator`：

```vb
Imports AbfScraper

' ── MyMemory（免费，无需 Key，国内可访问）──────────────────
Using t As New AbfTranslator()
    Dim zh = t.Translate("Deep groove ball bearing")
    ' 返回："深沟球轴承"
End Using

' ── 百度翻译（国内推荐，需在 fanyi-api.baidu.com 注册获取 Key）──
Using t As New AbfTranslator(
        provider:=TranslationProvider.Baidu,
        appId:="你的AppId",
        secretKey:="你的SecretKey",
        dictPath:="C:\MyApp\trans_dict.txt")   ' 可选，留空用程序目录
    Dim zh = t.Translate("Cylindrical roller bearing")
    ' 返回："圆柱滚子轴承"
End Using
```

**`AbfTranslator` 构造函数参数：**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `provider` | TranslationProvider | `MyMemory` | `MyMemory`=免费无需Key；`Baidu`=百度翻译 |
| `appId` | String | `""` | 百度 AppId |
| `secretKey` | String | `""` | 百度 SecretKey |
| `dictPath` | String | `""` | 词典文件路径，留空使用程序目录下的 `trans_dict.txt` |

> 翻译过的词条自动缓存到 `trans_dict.txt`，下次相同词汇直接命中缓存，不发网络请求。
> 缓存文件为 UTF-8 编码，Tab 分隔的两列（英文 → 中文），可手动编辑补充。

## 七、源码修改与编译

### 7.1 开发环境要求

| 工具 | 版本要求 | 下载地址 |
|------|----------|----------|
| .NET SDK | 6.0 或以上（推荐 8.0） | https://dotnet.microsoft.com/download |

安装后在命令行执行 `dotnet --version` 确认安装成功。无需安装 Visual Studio，命令行即可完成编译。

---

### 7.2 源码文件说明

源码目录结构：

```
AbfScraper/
├── AbfScraper.vbproj     项目文件（不需要修改）
├── AbfClient.vb          ★ 主要修改文件：登录、搜索、抓取逻辑
├── AbfTranslator.vb        翻译功能（MyMemory / 百度翻译）
├── BearingResult.vb        返回值数据结构定义（字段映射在此）
└── trans_dict.txt          翻译词典缓存（运行时自动维护）
```

**绝大多数需求只需修改 `AbfClient.vb`**，例如：
- 调整超时行为、分页逻辑、字段解析规则
- 修改搜索 URL 参数
- 调整返回值的组装方式

如需增减返回字段，同时修改 `BearingResult.vb` 中的字段定义和 `ToFields()` 方法。

---

### 7.3 编译步骤

在源码目录打开命令行，执行：

```
rd /s /q obj
dotnet build AbfScraper.vbproj -c Release
```

> 如果提示找不到 `obj` 目录可忽略第一条命令，直接执行第二条。

---

### 7.4 编译输出文件

编译成功后输出在：

```
bin\Release\netstandard2.0\
├── AbfScraper.dll        ★ 主库，给客户使用的文件
├── HtmlAgilityPack.dll   ★ 依赖库，必须与 AbfScraper.dll 一起提供
└── 其他文件              （不需要提供给客户）
```

**给客户时只需提供这两个 DLL 文件**，放在同一目录下即可。

---

## 六、注意事项

1. **网络要求**：需能访问 `https://www.abf.store`；使用百度翻译还需能访问 `https://fanyi-api.baidu.com`
2. **翻译限额**：MyMemory 每 IP 每日约 5000 字，超出后翻译返回空（字段保留英文），不影响采集数据
3. **百度翻译申请**：访问 `https://fanyi-api.baidu.com/` 注册开发者账号，创建应用后获得 AppId 和 SecretKey
4. **图片命名规则**：`型号字母数字全大写-结果序号-图片序号.扩展名`，如 `NU324ETVP2FAG-1-1.png`
5. **异常处理**：建议用 `Try...Catch` 包裹 `Login()` 和 `Search()` 调用
6. **资源释放**：使用 `Using` 语句确保 `AbfClient` 和 `AbfTranslator` 被正确释放
